GraphQL

A Graph Query Language

Making the Front End dumb by improving the queries

A GraphQL query asks the API for an specific structure

graphqlHTTP ({
    schema: buildSchema(`
        type Query {
            callback(numOne: Int!, numTwo: Int): Int
        }
    `),
    rootValue: {
        callback: function ({numOne, numTwo}) {
            return numOne + numTwo;
        }
    },
    graphiql: true,
})
